# @amlplugins/mistral-fim

Native `@mistralai/mistralai` SDK re-exported under the `@amlplugins` namespace.

> Mistral FIM (Fill-in-the-Middle) — code completion with prefix/suffix via Codestral.

## Install

```bash
npm install @amlplugins/mistral-fim
```

## Use

```ts
import { /* named exports from @mistralai/mistralai */ } from "@amlplugins/mistral-fim";
// or
import sdk from "@amlplugins/mistral-fim";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
